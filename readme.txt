Project: New issues viewer for Github

Javascript frameworks used: Jquery and underscore

API; https://api.github.com/repos/npm/npm/issues

API Documentation: http://developer.github.com/v3/issues/

Project brief: The purpose of this project was to create a new issues view for Github. Using there api to pull data and display them on a responsive web page. The project consist of two parts. The first part is to display the issue title, issue number, label, user name, user avatar, and the first 140 characters of teh issue ending on a clean word. The second part is once a user clicks on an issue it should open a new page with a detailed version of the issue. This page should include the issue title, state, labels, user name, user avatar, complete issue, and a list of comments if they exist.
